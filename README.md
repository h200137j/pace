## What is Pace?

**Pace** is a beautifully crafted, offline-first habit and activity tracker built with Flutter. It's designed around one idea: *making it frictionless to show up every single day*.

Whether you're building a 30-day coding challenge, tracking a daily meditation practice, or logging focused deep-work sessions тАФ Pace gives you the visual feedback and streak motivation to keep going.

No subscriptions. No cloud sync. No noise. Just you and your habits, stored privately on your device.

## тЬи Features

### ЁЯОп Three Activity Types
Create and track in three different modes depending on your goal:

| Type | Best For |
|---|---|
| **Challenge** | Time-boxed streaks тАФ "30 days of running" |
| **Task** | Recurring daily habits тАФ "Drink 2L of water" |
| **Focus** | Deep-work logging тАФ "2 hours of coding" |

Each activity gets its own **color**, **icon**, and **target day schedule** (e.g. weekdays only).

---

### ЁЯУЕ Frictionless Daily Check-In
Your home dashboard shows everything due today. A single tap on the animated progress ring marks a habit as **done** тАФ no menus, no extra steps.

- Live **streak badge** ЁЯФе updates instantly
- **Week dot indicator** shows your last 7 days at a glance
- **Gradient cards** pulse with your chosen activity color

---

### ЁЯФе Streaks That Keep You Honest
Pace calculates your streaks in real-time, with zero approximation:

- **Current streak** тАФ consecutive days done up to today
- **Longest streak** тАФ your personal record, tracked forever
- **Completion rate** тАФ percentage since you created the habit
- **Total completions** тАФ raw count of every day you showed up

---

### ЁЯУК Rich Analytics (fl_chart)

## ЁЯЪА Getting Started

### Prerequisites
- Flutter SDK `>=3.0.0`
- Dart `>=3.0.0`

### Installation

```bash
# Clone the repository
git clone https://github.com/h200137j/pace.git
cd pace

# Install dependencies
flutter pub get

# Generate Isar schemas (required on first run)
dart run build_runner build --delete-conflicting-outputs

# Run the app
flutter run
```

> **Note:** The Isar code generator (`build_runner`) must be run at least once to generate the `activity.g.dart` and `completion.g.dart` schema files. These are excluded from version control.
